## Installation

You can install the development version of spammR like so:

``` r
library(devtools)
devtools::install_github('pnnl-compbio/spammR',build_vignettes = TRUE,force=TRUE)
```

## Documentation

For complete documentation you can use the inline help in spammR or visit
[our documentaton site](https://pnnl-compbio.github.io/spammR/)
