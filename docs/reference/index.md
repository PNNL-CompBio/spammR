# Package index

## All functions

- [`calc_centers()`](calc_centers.md) : Calculate centers of
  spots/samples for distance based analysis

- [`calc_diff_pairwiseSamples()`](calc_diff_pairwiseSamples.md) : Helper
  function for computing pairwise differences between sample values

- [`calc_missingness()`](calc_missingness.md) : Calculate degree of
  missingness for each featurea and sample

- [`calc_spatial_diff_ex()`](calc_spatial_diff_ex.md) : Identify
  differentially abundant features across image

- [`convert_to_spe()`](convert_to_spe.md) :

  Create a `SpatialExperiment` object from data

- [`distance_based_analysis()`](distance_based_analysis.md) : Identify
  multiomic features correlated with an distance to an image feature

- [`enrich_gradient()`](enrich_gradient.md) : Calculate functional or
  pathway enrichment from a gradient

- [`enrich_ora()`](enrich_ora.md) : Calculate functional or pathway
  enrichment

- [`impute_spe()`](impute_spe.md) : Impute missing values based on
  spatial coordinates

- [`oneKSoilsCoords`](oneKSoilsCoords.md) : Soil sample coordinates for
  the sample information

- [`oneKSoilsMeta`](oneKSoilsMeta.md) : Soil samples from the 1000 soils
  project Row names indiciate KEGG Orthologs, Column names indicate
  locations Soil samples metadata for the sample information

- [`pancMeta`](pancMeta.md) : Here is the data to be included in the
  spammR package

- [`protMeta`](protMeta.md) : Protein metadata for general uniprot data,
  including the pancProts column for the panc data

- [`retrieve_metaspace_data()`](retrieve_metaspace_data.md) : Retrieve
  data objects from metaspace

- [`smallPancData`](smallPancData.md) :

  Pancreatic proteomics measurements from Gosline et al. proteomics
  study. Row names indicate individual proteins, columns indicate sample
  identifiers described in `pancMeta`. Individual values show
  un-transformed abundance values for each protein. List of proteomic
  measurements in a list sorted from individual regions of the pancreas.
  A subset of the data at
  https://figshare.com/articles/dataset/spammR_Files/29250962

- [`spat_reduce()`](spat_reduce.md) : Reduce a high resolution image
  measurement (origin) to a lower resolution measurement (target)

- [`spatial_heatmap()`](spatial_heatmap.md) : Plot data in heatmap
  together with image

- [`spatial_network()`](spatial_network.md) : Plot network of correlated
  features

- [`split_spe()`](split_spe.md) : Divide SPE object into smaller objects
  by column features

- [`volcano_plot()`](volcano_plot.md) : Create a volcano plot from
  differential expression data
